## SpringCloud都有哪些核心组件

SpringCloud有五大组件，现在主流的有两个体系，国内最常用的是SpringCloudAlibaba：

| 组件     | 阿里    | 网飞     |
| -------- | ------- | -------- |
| 注册中心 | Nacos   | Eureka   |
| 远程调用 | Feign   | Feign    |
| 负载均衡 | Ribbon  | Ribbon   |
| 服务保护 | Hystrix | Sentinel |
| 服务网关 | Gateway | Gateway  |
